Data Type	Description	Passport Area	Granularity	Example Use Cases
Market Size (Value)	Monetary measure of market size, expressed in current or constant currency terms.	Industries, Economies, Stats Grid	Country → Category → Year	Market sizing, trend analysis, forecasting, competitive benchmarking
Market Size (Volume)	Physical measure of market size, expressed in standard units (e.g., litres, tonnes, packs).	Industries, Stats Grid	Country → Category → Year	Consumption analysis, packaging trends, production planning
Company Share	Percentage of total market accounted for by a company within a category or geography.	Companies, Industries	Country → Category → Company	Competitive landscape, market entry strategy, M&A analysis
Brand Share	Percentage of total market accounted for by a brand within a category or geography.	Brands, Industries	Country → Category → Brand	Brand positioning, portfolio optimization
Price (RSP)	Retail Selling Price including VAT, quoted to consumers; used for pricing analysis.	Pricing, Via, Stats Grid	Country → Category → Pack Size	Price benchmarking, premiumization analysis, elasticity modeling
Channel Split	Breakdown of data by retail channel (e.g., on-trade vs off-trade).	Retailing, Industries	Country → Category → Channel	Distribution strategy, channel performance analysis
Currency Type	Indicates whether data is in current or constant terms; linked to inflation adjustments.	Stats Grid, Data Extractor	Global → Category → Year	Inflation-adjusted comparisons, historical vs forecast alignment
Inflation Type	Defines fixed vs year-on-year inflation adjustments applied to value data.	Stats Grid, API Parameters	Global → Category → Year	Economic modeling, real vs nominal growth analysis
Exchange Rate Type	Specifies conversion basis for multi-country comparisons.	Stats Grid, API Parameters	Global → Category → Year	Harmonized global comparisons, currency risk assessment
Forecast Data	Projected values or volumes for future years based on Euromonitor forecasting models.	Industries, Economies, Dashboards	Country → Category → Year (Future)	Strategic planning, scenario modeling, investment decisions
Historical Data	Actual recorded data for past years; forms the baseline for analysis and forecasting.	Industries, Economies	Country → Category → Year (Past)	Trend analysis, performance benchmarking
Attribute Measures	Data split by product attributes (e.g., health claims, packaging type, sustainability tags).	Attributes, Innovation, Sustainability	Country → Category → Attribute	Innovation tracking, product development, consumer trend analysis
Price Band Measures	Market size or share segmented into predefined price ranges.	Industries, Pricing Analysis	Country → Category → Price Band	Premium vs value segmentation, affordability studies
Rankings	Ordered lists based on size, share, or growth metrics.	Dashboards, Company Analysis	Country → Category → Metric	Competitive benchmarking, top performer identification
Percentage Measures	Derived metrics expressed as percentages (e.g., growth rates, penetration).	Stats Grid, Analysis Pages	Country → Category → Year	Growth analysis, penetration studies
Ecommerce Data	Online retail sales and share metrics, segmented by category and channel.	Retailing, Digital Commerce	Country → Category → Channel (Online)	E-commerce penetration analysis, omnichannel strategy
Innovation Data	Tracks new product launches, claims, and attributes across categories and geographies.	Innovation, Attributes	Country → Category → Launch/Attribute	Competitive intelligence, trend spotting, R&D prioritization
